## ⚡ 3 خطوات فقط للبدء

### 1️⃣ التحقق (1 دقيقة)

```bash
npm run verify:supabase
```

يجب أن ترى ✅ أمام جميع المتغيرات.

### 2️⃣ إنشاء الجداول (5 دقائق)

اقرأ تعليمات **`QUICK_START.md`** (الخطوة 2)

أو بسرعة:
1. افتح: https://app.supabase.com
2. SQL Editor > New Query
3. انسخ محتوى: `scripts/001_create_products_table.sql`
4. اضغط Run ✅

### 3️⃣ التشغيل (1 دقيقة)

```bash
npm run dev
```

ثم افتح: http://localhost:3000 ✨
